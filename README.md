Ornate Website Redesign 🚀
Maine Ornate company ki existing website ko ek naya aur modern look dene ke liye ise redesign kiya hai. Is project ka focus clean UI aur responsive layout par hai.

✨ Key Features
Modern UI/UX: User experience ko dhyan mein rakhte hue naya interface.

Responsive Design: Mobile, Tablet aur Desktop ke liye fully optimized (Bootstrap 5 ka use karke).

Interactive Components: JavaScript ka use karke smooth transitions aur interactive elements add kiye gaye hain.

🛠️ Tech Stack Used
Structure: HTML5

Styling: CSS3 & Custom Animations

Framework: Bootstrap 5

Functionality: JavaScript (ES6)
